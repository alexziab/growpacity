Installation
============

This page explains how to install the **growpacity** package.

Requirements
------------

- Python 3.10 or later
- NumPy
- Astropy
- Numba
- `OpTool <https://github.com/cdominik/OpTool>`_

Installation
------------

You can install the package via GitHub:

.. code-block:: bash

    git clone https://github.com/alexziab/growpacity.git
    cd growpacity
    pip install .
